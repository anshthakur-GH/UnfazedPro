# UnfazedPro Dashboard

Workforce Intelligence & Automation System.

## Setup Instructions

### 1. Backend Setup (FastAPI)
```bash
cd backend
pip install -r requirements.txt
python seed_demo.py           # Populates monitor.db with demo data
uvicorn main:app --reload --port 8000
```

### 2. Frontend Setup (React + Vite)
*Requires Node.js*
```bash
cd frontend
npm install
npm run dev
```

### 3. Open Dashboard
Navigate to `http://localhost:5173` in your browser.

## Project Structure
- `backend/`: FastAPI server with SQLite integration.
- `frontend/`: React + Vite SPA using Recharts for data visualization.
- `monitor.db`: Local database (generated by seed script).
